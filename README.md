# MedSync Pro: Sistema de Gestão de Saúde (Desktop)

##  Objetivos do Projeto
Este projeto foi desenvolvido como parte de uma atividade prática com foco em sistemas complexos. O objetivo principal foi criar um protótipo funcional de média fidelidade para desktop, priorizando:
* **Arquitetura da Informação:** Estruturação lógica de dados densos para facilitar a leitura.
* **Hierarquia Visual:** Destaque para informações críticas como horários e status de consultas.
* **Clareza Funcional:** Interface intuitiva para reduzir a carga cognitiva do usuário.
* **Flexibilidade de Navegação:** Garantia de múltiplas formas de realizar a mesma tarefa.

##  Caminhos de Navegação (Tarefa: Agendamento de Consulta)
Para atender aos requisitos de flexibilidade do briefing, o sistema permite realizar o agendamento de uma consulta através de dois caminhos distintos:

1. **Caminho Direto (Dashboard):** - O usuário utiliza o widget de "Ações Rápidas" localizado na tela inicial (Dashboard) para iniciar o agendamento de forma imediata.
   
2. **Caminho Exploratório (Menu Lateral):** - O usuário acessa a opção "Médicos" no menu lateral, filtra pelo profissional desejado e realiza o agendamento através do perfil detalhado do médico.

##  Tecnologias Utilizadas
* **Figma:** Design de interface e prototipação interativa.
* **GitHub:** Documentação e repositório do projeto.
* **Grid System:** Utilização de grid de 12 colunas para layout desktop (1440px).
