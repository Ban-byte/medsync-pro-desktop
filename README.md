# Portal Acadêmico: Sistema de Gestão Escolar (Desktop)

##  Objetivos do Projeto
Este projeto consiste no desenvolvimento de um protótipo funcional de média fidelidade para um sistema de gestão acadêmica desktop. O foco foi a organização de sistemas complexos, priorizando:
* **Arquitetura da Informação:** Estruturação de dados densos (CR, disciplinas, créditos e prazos).
* **Hierarquia Visual:** Uso de cards e cores para destacar o status do período de matrícula.
* **Clareza Funcional:** Interface limpa para facilitar a gestão da vida acadêmica pelo estudante.
* **Flexibilidade de Navegação:** Implementação de dois fluxos distintos para a conclusão da tarefa principal.

##  Caminhos de Navegação (Tarefa: Realizar Matrícula)
Para atender aos requisitos de flexibilidade do briefing, o sistema permite realizar a matrícula em disciplinas através de dois caminhos distintos:

1. **Caminho Direto (Matrícula Rápida):** - Através do widget "Matrícula Rápida" no Dashboard principal, o usuário visualiza disciplinas sugeridas com base em seu histórico e realiza a matrícula com um clique no botão "Matricular Agora".
   
2. **Caminho Exploratório (Grade Curricular):** - O usuário acessa o menu lateral "Grade Curricular", onde pode navegar por toda a estrutura do curso, analisar dependências e pré-requisitos antes de selecionar a disciplina para matrícula.

##  Tecnologias Utilizadas
* **Figma:** Design de interface e prototipação de média fidelidade.
* **GitHub:** Documentação e versionamento do projeto.
* **Layout Grid:** Estrutura baseada em grid de 12 colunas para otimização desktop.
